# Desktop Island — Interactive Mockup

**"Your desktop's nervous system"**

An interactive HTML mockup of Desktop Island v7 — a Mac ambient HUD built through 5 rounds of synthetic user research (140 personas, 3,136 model evaluations).

## Open it

Just open `index.html` in any browser. No install, no server needed.

Or view live: https://noatsbot.github.io/desktop-island/

## What's interactive

- **Click any block** (Focus, Weather, Markets, News) to expand its peek card
- **Hit + button** to open the card picker (tiered by setup cost)
- **Use state buttons** at the bottom to switch: Focus / Meeting soon / Now Playing / Idle
- **⌘K** opens the command palette

## Key design decisions (v7)

| Decision | Finding |
|----------|---------|
| Focus Timer leads | 7/28 install votes — #1 by wide margin |
| No auto-rotation | Killed 80% of install intent at v1 |
| "nervous system" hook | 18/28 — runner-up got 5 votes |
| Zero OAuth | 0/28 would authenticate before experiencing value |
| Sports = opt-in | Alienates 8-10 personas as a default |
